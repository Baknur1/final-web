# Warehouse Management System

A full-stack modular warehouse management system with RBAC security.

## Stack
- **Backend:** Node.js, Express.js
- **Frontend:** Vanilla HTML, CSS, JS
- **Security:** JWT, Bcrypt, RBAC

## Roles
1. **Super Admin:** Global control, warehouse management.
2. **Manager:** Warehouse supervisor, staff management.
3. **Worker:** Operator, inventory status updates.
4. **Supplier:** Client, creating item requests.

## Setup
1. `npm install`
2. `npm start`
3. Access at `http://localhost:5000`

## Docker
1. `docker-compose up --build`
